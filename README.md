# Leyre clone (por características)

Este proyecto reconstruye una app web tipo landing/portfolio con funcionalidades comunes:

- Hero con CTA
- Sección de servicios con tarjetas
- Timeline de proceso
- Formulario de contacto con feedback en cliente

## Ejecutar

```bash
python3 -m http.server 8000
```

Luego abre `http://localhost:8000`.

## Nota legal

No se extrajo código fuente propietario de terceros. Esta implementación es una recreación de estilo y funcionalidades generales.
